Why use Adelphos Chat

Adelphos Chat runs in any modern browser and as a native Windows / macOS desktop app. It carries the same project memory the Revit Copilot uses — when you log in, it already knows which project you're working on, what you discussed yesterday, what's in the brief, what's in the spec, and what was last issued.

From the chat you can query the live Revit model (read-only when off-Revit, read-write when connected), pull figures from your calc spreadsheets, draft client-ready text, search project minutes, and trigger any tool the platform exposes.

For a project lead this typically reclaims 3 to 5 hours per week previously lost to context switching: walking back to the desk to look something up, opening Revit just to read a schedule, drafting the same status update three times across email, Teams and meetings. It also removes the daily "can someone send me…" pings to the modelling team because the agent can answer them instantly with the live data.

Your project knowledge, available anywhere. On site, in a meeting, on a train — the answer comes from the live model, not a phone call back to the office.

How it works

On site, in a meeting, on a train. Need a number from the model, a quote from the brief, a check against the spec — and you can't open Revit.

Ask Adelphos. It already knows the project. It pulls the answer from the live model, the brief, the spec, the schedule. You move on.

What makes it different

Most AI chat tools are stateless and project-blind. Adelphos Chat is built around your projects: it remembers what you discussed yesterday, knows which job you're on, and can act on the live data, not a snapshot.

Adelphos Chat features

AEC specialist on tap Trained on MEP, BIM, Building Safety, RIBA stages, ACC and CIBSE/BSRIA guides — answers like a senior engineer, not a generalist chatbot.
Drafts MEP specifications Authors Stage 2/3/4 specs from the live Revit model, brief and your house manufacturer library.
Builds and validates schedules Generates radiator, AHU, fan, light and fixture schedules straight from the model — and gap-checks them against your calcs.
Drafts client documents Status updates, scope notes, meeting minutes, site reports, survey writeups — formatted to your house style.
Live model query (Bridge on) Pull a number, parameter, view or schedule row from the live Revit model in seconds.
Reads project folders Briefs, specs, schedules, calc spreadsheets, prior drawings — all ingested and searchable.
Files incoming files Drag in PDFs, surveys, photos, emails — auto-filed against the right project, with the right metadata.
Project memory Remembers what you discussed yesterday, which job you're on, and every decision logged so far.
Search the project decision log Every previous answer, decision and rationale is searchable — the project's institutional memory.
