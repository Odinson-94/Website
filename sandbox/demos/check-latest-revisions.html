Check Latest Revisions on Site

Document Control Live 1:09

Compare site-issued PDFs with current model revisions; flag stale.

What it does

Check Latest Revisions on Site is part of the Document Control set in BUILD MEP. Compare site-issued PDFs with current model revisions; flag stale. The video above shows the workflow end-to-end against a sample model.

How it works

  1. Open the chat panel and ask "check latest revisions on site"
  2. BUILD MEP shows what it's about to do, then runs it against the active model
  3. Review the result inline; commit, undo, or refine with a follow-up prompt

What you'll see in the chat

BUILD MEP narrates each step as it happens. Pre-prompts, thinking steps, success and failure prompts are all defined per command (see the command documentation for an example of the underlying structure).
