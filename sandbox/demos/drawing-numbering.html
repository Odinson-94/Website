Drawing Numbering

BIM Live 1:08

Re-number every sheet against the project numbering convention.

What it does

Drawing Numbering is part of the BIM set in BUILD MEP. Re-number every sheet against the project numbering convention. The video above shows the workflow end-to-end against a sample model.

How it works

  1. Open the chat panel and ask "drawing numbering"
  2. BUILD MEP shows what it's about to do, then runs it against the active model
  3. Review the result inline; commit, undo, or refine with a follow-up prompt

What you'll see in the chat

BUILD MEP narrates each step as it happens. Pre-prompts, thinking steps, success and failure prompts are all defined per command (see the command documentation for an example of the underlying structure).
