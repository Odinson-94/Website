Dismiss All Revit Warnings Of Same Type

writes to model Public API · not yet exposed

Dismisses all Revit warnings matching a specified failure definition type. Use MCP.GetRevitWarningsByFailureType to identify the type first. Requires user confirmation — never auto-execute without explicit approval.

Before you run this

Active document must have warnings of the specified type.

After it runs

All warnings of the specified type are dismissed.
