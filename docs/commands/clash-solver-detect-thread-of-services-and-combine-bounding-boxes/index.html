Decision flow

1) Resolve input ids: Resolver_ElementIds (Bot) else active selection (Human).
2) WireError if fewer than 2 element ids (nothing to thread).
3) angle_tol_deg = Resolver_AngleTolDeg ?? 5.0; lateral_tol_mm = Resolver_LateralTolMm ?? 300.0.
4) Call CombineBoundingBoxesSafely.Combine(doc, ids, angle, lateral).
5) Serialise threads + unthreaded into tracker outputs.
6) Human path: TaskDialog summary "N threads, X unthreaded".
