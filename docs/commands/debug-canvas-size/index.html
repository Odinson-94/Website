Debug Canvas Size

General writes to model Public API · not yet exposed

Debug command: finds Revit's canvas panel window handle and dimensions. Enumerates all child windows of the main Revit window to identify the drawing canvas by size, visibility, and aspect ratio match against the active Autodesk.Revit.UI.UIView. Uses Adelphos.Revit.AIChat.Services.Execution.CommandFeedback for output.

Before you run this

Active Revit document must be open with a valid active view.

After it runs

Debug log written to canvas_debug.txt. Results fed back via CommandFeedback.

When to use this

  • Developer needs to identify the canvas window handle for screen-to-model coordinate mapping.
Don't use this when

Production workflows. Modifying the model. End-user features.
