Extend Pipe Down

Drainage writes to model Public API · not yet exposed

Command: Extends a selected pipe downwards by radius × 5. Uses ExtendPipeDown for the core extension logic. Related: ExtendFromConnectorCommand for connector-based extension. WORKFLOW: 1. User selects a pipe 2. Command gets pipe properties (system, size, routing type) 3. Delegates to ExtendPipeDown.Extend to create extension 4. Reports success/failure via CommandFeedback

Before you run this

Active document must be open. At least one pipe must exist in the model. User must select a valid pipe element.

After it runs

A new pipe is created extending downwards from the bottom of the selected pipe. Extension length = pipe radius × 5. New pipe matches system, size and family routing types.

When to use this

  • User needs to extend a drainage pipe downwards.
  • User wants to quickly add a below-slab extension.
  • User is extending vertical stacks through multiple floors.
Don't use this when

Not for autonomous AI execution. This command requires interactive Revit UI input (dialogs, picks, or manual confirmation).

Side effects

Creates a new pipe element in the model.
