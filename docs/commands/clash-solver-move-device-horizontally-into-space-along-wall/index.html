Before you run this

Active document; device is FamilyInstance with Wall host.

After it runs

Zero or one MoveElement inside inner Transaction.

When to use this

  • Slide wall-hosted device to new position along the same wall.
Don't use this when

Off-wall moves — use MOVE_TO_HOST or MOVE_TO_WALL.

Inputs

NameKindTypeRequiredDefaultDescription
PresetKind config string no "clash_tolerances" Deterministic config preset

Decision flow

1) device unique id required OR single selection.
2) Resolve device; WireError if not FamilyInstance with Wall host.
3) wall = fi.Host as Wall; tangent = wall.LocationCurve tangent direction (projected onto XY).
4) If prefer forward=false, flip tangent.
5) Delegate TryDirectionalMoveOnService.ProbeAndMove(doc, fi, tangent, step_ft, threshold_ft, centreFallback=false).
6) Emit outcome + wall_tangent_ft.
