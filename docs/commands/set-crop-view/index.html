Set Crop View

General writes to model Public API · not yet exposed

Allows user to draw a rectangle corner-to-corner to set as the view's crop region. Uses SetCropView.FromCorners for the crop region geometry calculation. Only works on plan, section, and elevation views — not on sheets, schedules, or 3D views.

Before you run this

Active view must be a plan, section, or elevation view (not sheet, schedule, or 3D).

After it runs

View crop region is set to the rectangle defined by the two picked corners. Crop is enabled.

When to use this

  • User wants to define a custom crop region on a plan or section view for sheet placement.
Don't use this when

Cropping 3D views or sheets. Resetting crop to default. Scope box assignment (use SetViewToScopeBoxCommand).
