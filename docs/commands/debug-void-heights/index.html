Debug Void Heights

General writes to model Public API · not yet exposed

Debug command for testing ceiling void height calculations. Picks a point and displays all related measurements: room detection via DetectRoomFromServiceStartPoint, ceiling via DetectCeilingHeightFromRoom, slab via DetectSlabAboveHeight, void depth via CalculateVoidHeightFromCeilingAndSlab, centre-of-void via CalculateMiddleOfVoid, and 3D raycast via RayTrace3D.

Before you run this

Active document must be open with a valid plan or 3D view.

After it runs

Debug results are displayed via CommandFeedback. No model changes.

When to use this

  • User wants to verify void-height calculations at a specific point.
Don't use this when

Production use. Use CalculateMiddleOfVoid directly for runtime logic.

Returns

NameTypeDescription
VoidDepthstringVoid depth in mm
CenterHeightstringCentre-of-void height from floor in mm
