Select By Room

writes to model Public API · not yet exposed

Returns elements inside an architectural room (by number or name). Optionally sets Revit selection to those elements. Uses GetElementsInRoomHelper.ResolveRoom to find the room and GetElementsInRoomHelper.GetElementsInRoom to collect elements within it. Called by MEPBridge.Revit.AIChat.Services.CustomTools.SelectByRoomTool via execute_command. Related: SelectByIdsCommand for ID-based selection, SelectByLocationCommand for coordinate-based selection.

Before you run this

Active document; roomNumber or roomName required.

After it runs

Returns { success, count, room, items }; optionally selection set.

When to use this

  • List/filter elements in a room; then use get_selection or SelectByIdsCommand for cross-off.
Don't use this when

Selecting by coordinates — use SelectByLocationCommand. Selecting by ID — use SelectByIdsCommand.
