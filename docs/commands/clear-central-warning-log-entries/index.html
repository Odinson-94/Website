Clear Central Warning Log Entries

writes to model Public API · not yet exposed

Clears all entries from CentralWarningLogger. This only affects the in-memory log — persisted JSON and Supabase entries are not modified. Requires user confirmation before execution.

Before you run this

Central warning log should have entries to clear.

After it runs

In-memory warning log is empty.
