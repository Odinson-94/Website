Calculation engines

Verified calculation modules the agent calls during MEP design. Each engine runs server-side and returns auditable results — the agent never invents a number, and you can trace every result back to the inputs that produced it.

Each engine is invoked through the Revit Copilot or Adelphos Chat with auditable inputs. Outputs are versioned and traceable to the model element, brief clause or system that drove them.
