Clash Solver

Reports clashes. Resolves them — with your permission. Logs every decision.

Connect Clash Solver to your ACC project. It pulls the latest clash report, deduplicates against the previous run, and triages: real clash vs ghost (model-link drift, family swap, view-filter mismatch). For each real clash it proposes a resolution — re-route the smaller service, raise the ceiling void in the affected zone, swap a fitting, etc. — and shows you what the model would look like after.

You accept, refine, or reject each proposal. Set it to auto-resolve when the agent is confident enough, or keep full manual control. Every resolution is applied as an auditable Revit transaction. Every change records who made it, which agent proposed it, what was changed, and why — the full audit trail.

CoBie Manager

A live COBie deliverable, never a panic before hand-over.

CoBie Manager is the COBie tool every BIM coordinator wishes Revit shipped with. Populates 20+ COBie sheets from the live model, validates against BS 1192-4 and your project EIR, and tracks every change with an audit log.

Every cell traces back to the source Revit element. No manual mapping, no copy-paste from schedules — the agent reads the model, fills the workbook, flags what's missing, and hands you a file ready for the client.
