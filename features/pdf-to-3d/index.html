The AI reads blocks and symbols from the source drawing and maps them directly to your Revit model — no manual configuration, no mapping tables, no family-by-family setup.

Send a 2D PDF or DWG. The AI recognises every block, symbol and line convention in the drawing, understands what each one represents, and maps them to the correct families in your Revit template. Radiators, grilles, sockets, luminaires, sanitary fittings — all identified and placed automatically using AI, not lookup tables.

Where the source drawing is ambiguous — annotation overlap, unclear connections, missing system labels — the AI flags it, proposes a resolution, and asks you to confirm.
